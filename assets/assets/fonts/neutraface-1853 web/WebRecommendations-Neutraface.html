WEB RECOMMENDATIONS

Here is our recommendation for code to use in your css stylesheet when using the House Industries web fonts. Just copy and paste this code into your style sheet, replacing Neutraface Text with the name of the font. Then apply to desired rule, as shown below applied to an h3 tag. We recommend using text-rendering: optimizeLegibility; in order to ensure that kerning is applied correctly in all browsers.

@font-face {
    font-family: "Neutraface Text";
    src: url("NeutrafaceTextBook.woff2") format("woff2"),     url("NeutrafaceTextBook.woff") format("woff");
    font-style: normal;
    font-weight: 400;
    font-stretch: normal;
}

Replace Neutraface Text with a name of your choosing The next line should contain the woff2 filename, and location, if in a subdirectory. The following line should contain the woff filename, and location, if in a subdirectory. Note that if the fonts are in a subdirectory, that location must also be specified.

If you are using each font separately, use the style and weights shown. If the fonts should behave as part of a family, see notes below.

h3 {
    font-family: "Neutraface Text", sans-serif;
    font-weight: normal;
    font-kerning: normal;
    font-variant-ligatures: common-ligatures, contextual;
    -moz-font-feature-settings: "kern", "liga", "clig", "calt";
    -ms-font-feature-settings: "kern", "liga", "clig", "alt";
    -webkit-font-feature-settings: "kern", "liga", "clig", "calt";
    font-feature-settings: "kern", "liga", "clig", "calt";
}

Setting the font weight to normal in a Header tag will make sure you don't get faux bolding added.

The kerning and ligature features should always be turned on. The browser variants are currently required.

'c2sc' = Capitals to Small Caps
'case' = Upper Case
'dnom' = Denominators
'frac' = Fractions
'lnum' = Lining Figures
'numr' = Numerators
'onum' = Old Style Figures
'salt' = Alternate G, M, Q, a, g, u and f ligatures
'sinf' = Inferiors
'smcp' = Small Caps
'sups' = Superiors
'tnum' = Tabular Figures

These features are available in the Neutraface Text fonts. If needed they can be inserted into the font-feature-settings statements.

SETTING UP FONT FAMILIES

If you are setting up a font family, and wish the <i> tag to show the italic version of the font, you need to set the weights and styles correctly. In this case, you need an @font-face section for each font, but set the font-style and font-weight like this:

(for roman font)...
font-weight: normal;
}
(for italic font)...
font-weight: normal;
font-style: italic, oblique;
}
(for bold font)...
font-weight: bold;
font-style: normal;
}
(for bold italic font)...
font-weight: bold;
font-style: italic, oblique;
}
